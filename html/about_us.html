About CNR Farm Web App

CNRFarm is a farm record web application that runs on a web browser, utilized for smart and effective record keeping of livestock activities and enterprises.
This webapp is created through interlinked functionings of HTML, CSS, Firebase and JavaScript. Unlike physical book recordings, it stores and secures data
quality with strict privacy standards by firebase; a secure real-time database. It comprises three major livestock activities viz. Dairy, Piggery & Poultry, covering
wide range of features such as livestock censuses, production statistics, health statuses and livestock populace.

How it works?

  •

    Registration

    The user should register for farm recording entry by signing up by filling up necessary information
    After the sign-up, the webpage will direct you to the dashboard that will have three categories of livestock production.

  • Data Entry

    Clicking on one of the categories will redirect the user to another webpage where the user can enter their data on livestock activities.
    Individual categories will have different features and content entry functions for livestock data documentations
